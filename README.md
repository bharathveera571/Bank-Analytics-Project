# Bank Analytics Project  

## Overview  
This project focuses on analyzing **loan data** using **Excel, Power BI, SQL, and Tableau** to identify key insights such as **loan trends, borrower risk assessment, and financial decision-making**. By leveraging these tools, the project provides **data-driven recommendations** to improve loan management.  

## Tools Used  
- **Excel** – Dashboards, Pivot Tables  
- **Power BI** – Interactive Reports  
- **Tableau** – Data Visualizations  
- **MySQL** – Data Queries & Key Performance Indicators (KPIs)  

## Files Included  
- **Bank Analytics Final PPT.pptx** – Project Presentation  
- **Bank Analytics Power BI Dashboard.pbix** – Power BI Dashboard  
- **Final Tableau Dashboard.twbx** – Tableau Dashboard  
- **Excel Dashboard Final.xlsx** – Excel Dashboard  
- **Bank Analytics Project.sql** – MySQL Queries  
- **Finance_1.csv, Finance_2sql.csv, Main Data.xlsx** – Data Files  

## Insights & Recommendations  
- **Identified high-risk loan profiles**, enabling better risk assessment.  
- **Suggested debt consolidation strategies** to optimize financial planning.  
- **Built predictive models** to analyze loan repayment behaviors.  

## Conclusion  
The **Bank Analytics Project** provides valuable insights into **loan trends and financial risks**, helping financial institutions and lenders make **data-driven decisions** for improved **loan management and risk assessment**.
